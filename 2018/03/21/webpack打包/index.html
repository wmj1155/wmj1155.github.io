使用webpack工具打包js和css文件

很多时候我都在纠结要用什么来压缩和打包自己的js和css文件，特别在未使用任何框架的时候，就需要我们自己利用工具来打包，这次我选择了webpack作为打包和压缩的工具。

1.首先需要新建一个项目，例如项目名为app

2.在改目录下输入npm init生成package.json文件

3.安装webpack模块 输入npm install –save-dev webpack

4.安装loader 输入npm install –save-dev css-loader 和npm install –save-dev style-loader

5.在app目录下新建styles目录，并在styles目录下新建index.css文件；新建scripts文件夹，并在其文件夹下新建index.js文件和entry.js(作为入口文件)

6.在app文件夹下新建webpack.config.js用于配置

7.在index.html中引入最后生成的bundle.js

8.在package.json文件中配置命令”webpack”:”webpack”

9.最后在命令行中输入npm run webpack即生成bundle.js
